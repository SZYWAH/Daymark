# Daymark 0.1.0-rc.5 发布说明

Daymark 是一个本地优先的个人知识库桌面应用，用来记录日常内容、整理资料、沉淀长期记忆，并在用户授权的范围内使用 AI 辅助整理和回顾。

## 版本状态

- 版本：0.1.0-rc.5
- 平台：Windows x64
- 安装包：NSIS `.exe`
- 状态：发布候选版，适合小范围试用和反馈

## 安装与安全提示

- 双击安装包后按安装向导完成安装。
- 可以直接覆盖安装此前 RC 版本，无需先卸载；建议升级前导出一次核心备份。
- 当前版本未做代码签名，Windows 可能显示安全提醒；这是未签名应用的常见提示。
- 请从本仓库的 GitHub Release 下载，并可使用发布页提供的 SHA256 校验值验证文件完整性。

## RC5 主要变化

- 全新空库首次启动时提供 9 条精简示例资料和 5 个目录，展示收件箱、整理状态、待读、收藏、摘要与待办；可在设置中安装或安全删除。
- 正式示例使用固定标识并排除在核心备份之外；用户放入示例目录的真实资料和必要目录链仍会保留。
- 资料库桌面端改为固定窄全局栏与可折叠目录栏，父目录数量和筛选范围包含全部后代资料。
- 目录层级、树形引导、目录操作菜单和资料 AI 操作的可发现性进一步统一。
- Windows 最小化后保留任务栏入口，关闭按钮继续隐藏到托盘，托盘“退出”真正结束程序。
- 顶部快速记录支持左、中、右三档停靠，改善自动收回稳定性，并采用 Enter 保存、Shift+Enter 换行。

RC5 不升级 IndexedDB 数据结构，不改变应用标识符和数据目录，也不会自动调用 AI。

## 现有核心能力

- 本地日志、资料库、长期记忆、内容关联与全文搜索。
- 核心数据备份和覆盖恢复。
- 桌面端 API Key 系统凭据存储，并按供应商与 Base URL 隔离。
- Codex 与 Claude Code 会话只读扫描、正文预览、AI 回顾和回顾归档。
- 用户主动开启后的自动工作回顾，以及顶部快速记录悬浮窗。

## 数据与隐私

- Daymark 的内容保存在用户本机，不会主动上传到服务器。
- 桌面端手动填写的 API Key 保存到系统凭据存储。
- Web 模式仅作为本机开发降级路径，不建议用于共享设备。
- 整理、总结、回顾和提炼通常只在用户主动触发时调用 AI。
- 用户明确开启“自动工作回顾”后，Daymark 会先在本机对新增会话正文脱敏，再按固定周期发送给用户配置的 AI 服务；关闭后不再自动读取正文或调用 AI。

## 备份范围

- 核心备份包含资料、目录、日记、长期记忆文档、记忆卡片和内容链接。
- 核心备份不包含 API Key、AI 设置、AI 草稿、自动工作回顾草稿、历史总结报告、会话索引、主题或布局偏好。
- Daymark 内置示例资料及仅供示例使用的空目录不进入核心备份。
- 恢复核心备份会覆盖当前核心内容，恢复前请确认备份文件来源可信。

## 反馈建议

反馈问题时请说明 Windows 版本、Daymark 版本号、发生问题的页面和操作步骤；如有报错，请仅提供脱敏后的错误信息。
